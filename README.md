## About

Cubewire is an enterprise-grade Web3 platform that provides secure, scalable infrastructure for building next-generation payment and digital asset applications. Developed by [p8.io](https://p8.io), Cubewire delivers institutional-grade wallet infrastructure, multi-asset portfolio management, and a public L2 chain — all backed by comprehensive security practices.

## Products

### Wallet-as-a-Service

Create and manage institutional-grade vaults, sign transactions, and handle multi-asset portfolios with secure wallet infrastructure.

- **Vault Management** — Create and manage secure vaults
- **Multi-Asset Support** — Handle diverse digital asset portfolios
- **Transaction Management** — Send, track, and manage transactions
- **Smart Contracts** — Deploy and interact with smart contracts
- **Policy Engine** — Configurable approval policies and permissions
- **Audit Logging** — Comprehensive audit trail for all operations
- **Webhook Notifications** — Real-time event notifications

### Public L2 Chain

A high-performance Layer 2 blockchain built for enterprise payment workloads.

## Security

Security is at the core of everything we do.

- All data encrypted at rest using HSM-backed keys via cloud KMS
- Application secrets encrypted and stored via Secrets Manager
- Annual penetration testing by leading security firms
- Continuous vulnerability scanning across the SDLC
- 24/7/365 monitoring and endpoint protection
- SOC 2 and Australian Privacy Principles (APP) compliance

Trust Center: [https://trust.p8.io](https://trust.p8.io)
